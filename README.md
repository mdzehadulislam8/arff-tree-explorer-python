# ARFF Tree Explorer

A simple and interactive Python application for loading ARFF datasets, comparing ML classifiers, and visualizing decision trees. Built using Tkinter, scikit-learn, and matplotlib, this tool helps students and beginners explore machine learning models in a clear and intuitive way.
